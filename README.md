# Insurance-Data-Analysis-
Data Science project analyzing medical insurance costs using EDA, visualization, outlier detection (IQR), Simple Linear Regression, and hypothesis testing (T-Test) to understand the impact of factors like BMI, age, and smoking.
# Insurance Data Analysis using Linear Regression

## 📌 Project Overview
This project analyzes medical insurance data to understand how factors like age, BMI, and smoking affect insurance charges.

## 🔍 Features
- Exploratory Data Analysis (EDA)
- Correlation & Covariance
- Outlier Detection (IQR Method)
- Data Visualization
- Simple Linear Regression
- Hypothesis Testing (T-Test)

## 🛠 Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- SciPy

## 📊 Key Insights
- Smokers have significantly higher charges
- BMI and Age impact insurance cost
- Strong relationships identified using heatmaps
